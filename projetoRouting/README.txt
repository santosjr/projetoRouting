Component: Routing

	O Routing é um componente do symfony que é usado para otimizar as rotas do sistema web desenvolvido e torná-las mais límpidas, de modo a permitir que o desenvolvedor crie suas próprias rotas, evitando a exibição, por exemplo, do nome dos arquivos e deixando seu site bem mais intuitivo para o usuário final.
	Nesse Pojeto desenvolvido buscou-se usar do Routing para definir as rotas do sistema web, em que, para isso, as mesmas foram configuradas no arquivo chamado pageController, através de comandos padrões do symfony para se designar uma rota para determinada página web. A finalidade desse proeto é exemplificar, de forma bem simples e prática, o uso do routing em aplicações web.
